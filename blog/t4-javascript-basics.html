How does JavaScript compare to HTML and CSS?

It's harder to grasp but also more useful. It has greater functionality and complexity, and is concenred with the interative aspects of a website / application.

Explain control flow and loops using an example process from everyday life

A control flow is the like the steps you need to wash the dishes, a checklist of tasks that allow you to finish washing them. A loop is like repeating those steps after each meal.

Explain the difference between accessing data from arrays and objects

You access data within an array by using an index number. You access data within an object by using a name.

Explain what functions are and why they are useful

Functions are a block of code that perform a task when they are called. They are useful because you can reuse the code as much as you need to, in order to produce different (or similar) results.
