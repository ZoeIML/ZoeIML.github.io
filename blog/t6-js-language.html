In the JavaScript language, describe all of the use cases for each of the following:

Parentheses ()

Parentheses contain parameters for functions and/or group expressions together. For example: function() has no parameters but the parentheses are needed to tell the computer that it's a function - like an activate switch. On the other hand function(cells) relates to cells (on a table, let's say) and gives the function a parameter to work in. Then there's something like: for (let i = 0; i < board.cell.length; i++), which tells the computer what to do and in what order.

Brackets []

Brackets are used to create a list, which is called an array in JavaScript.

Braces {}

Braces are used in either functions (which are like an action the computer is going to take) or objects (i.e., a game board). Mostly, they help declare variables (make them clear) or create objects.

Single Quotes ''

Words and sentances, known as strings, go in single quotes to tell the computer to treat them as words.
