What is scope, and how does it work in JavaScript?

In JavaScript, there are items called variables. A variable might be a number, a group of numbers, a group of words, or a set of data (to name some). Scope refers to what kind of variable (or thing) it is. A global variable is a thing that stays the same throughout the whole document. A local variable is a thing that changes depending on what you're doing (what function you're asking the computer to perform).
