What is a responsive site, and why is it important?

A responsive site changes according to the size of the screen it is being viewed on. It is important because it ensures all users have a positive experience interacting with the website.

What is mobile first design is, and why is it important?

Mobile first design is important because it focuses on mobile or small screens first and then builds up. This means the mobile site is as enhanced as possible for small devices, which is important because of the number of users who are now viewing sites on smaller devices such as phones.

What are frameworks? What are their pros and cons?

A framework is standardised code used to support website development. They tend to address specific issues, such as layout. The pros include efficiency and support. The cons of frameworks include learning the framework rather than the language and limitations to the core behaviour of the framework.

What is a wireframe and why do we use them?

A wireframe is a visual representation of elements on a webpage. We use them as blueprints for websites, which allow us to simplify communication and create a hierarchy of information.
