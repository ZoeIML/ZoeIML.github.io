What is grid based design?

A tool used to organise a webpage using rows and columns in order to form a grid.

What's all the hype about responsive pages?

A responsive page is one that responds to different screen resolutions but changing size. It's important because people use different devices and screens, and have a responsive page means that all users have the same experience.

What is semantic structure?

It communicates the layout of content on a webpage to the web browser. I.e., if it's an image, heading, etc.
