Can you think of an analogy to describe HTML, CSS and the DOM? How would you describe it to a non-technical friend?

  • HMTL is like the physical elements of a boardgame: cards, board, dice, etc.
  • CSS is like the design / look of the game: colours, themes, images, etc.
  • DOM is like the rule book for a boardgame in that it tells you how to interpret the elements of it in order to be able to play.

What is meant by boxifying design?

Dividing the elements of the webpage design into boxes. The page is like a big box that houses all of the smaller ones.

What is the box model?

The box model helps with understand what a single box on the webpage contains in terms of format. The four parts, starting from the middle and working outwards, are:

  • Content: The nucleus of the box, which is surrounded by the other parts
  • Padding: The area around the content that directly protects it
  • Border: The space around the content and padding
  • Margin: The transparent space around all of the above, which is essentially the space between a box and other boxes on the page
